# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2024-12-16

### Added
- Enterprise-ready pip package structure
- MIT License
- MANIFEST.in for proper package distribution
- pyproject.toml for modern Python packaging (PEP 517/518)
- Comprehensive documentation for pip installation
- CI/CD workflows for automated PyPI publishing
- Desktop GUI application with native interface
- Streamlit web interface
- Command-line interface (CLI) with comprehensive options
- Packet capture and analysis (Wireshark-like features)
- Port scanning capabilities (nmap-like features)
- Network discovery and mapping
- DNS and WHOIS tools
- Network information and monitoring
- Network traffic visualization (EtherApe-like features)
- Real-time bandwidth monitoring
- Multi-threaded scanning for performance
- Service detection with banner grabbing
- Protocol parsing and analysis
- PCAP file export functionality
- Anomaly detection in network traffic
- Top talkers analysis
- Conversation tracking

### Changed
- Updated project structure for pip distribution
- Enhanced setup.py with comprehensive metadata
- Improved README with installation and usage instructions
- Updated .gitignore for better build artifact handling

### Security
- Added security disclaimer for authorized use only
- Implemented proper privilege handling for packet capture

## [1.0.0] - Initial Release

### Added
- Basic network packet analyzer
- Simple packet capture functionality
